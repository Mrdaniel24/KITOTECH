# Marketing Site — UI Kit

The public-facing Kitotech.com kit. A single-page composition demonstrating: sticky glass navigation, cinematic hero with live ticker, product showcase, capabilities grid, customer logo wall, CTA strip, footer, plus floating WhatsApp + AI assistant launchers.

This is a **visual recreation only** — buttons, the AI assistant, and WhatsApp open inert demonstration panels, not real services.

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Page shell. Loads React, Babel, the design-system CSS, and every component. |
| `Nav.jsx` | Sticky top bar — becomes glass on scroll. |
| `Hero.jsx` | Cinematic hero with eyebrow, headline, live channel counter. |
| `ProductShowcase.jsx` | Three product cards (Sentinel, Gate Vision, Access OS). |
| `Capabilities.jsx` | Four-up capabilities grid with hairline-only treatment. |
| `Stats.jsx` | Numeric strip — uptime, channels, latency, sites. |
| `LogoStrip.jsx` | Customer logo wall (wordmarks). |
| `CTAStrip.jsx` | Dark CTA section, red primary button. |
| `Footer.jsx` | Multi-column footer + small print. |
| `Floaters.jsx` | WhatsApp bubble + AI assistant launcher + panels. |
| `Icon.jsx` | Inline icon component (subset of Lucide). |

## Open `index.html` to see the kit.
